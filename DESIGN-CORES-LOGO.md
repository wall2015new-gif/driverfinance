## 💡 DICAS DE USO

### Para Manter a Consistência:
1. Use sempre as variáveis CSS (--logo-green, etc.)
2. Não use cores hardcoded
3. Mantenha os glows sutis
4. Use gradientes do logo quando possível

### Para Adicionar Novos Elementos:
1. Escolha uma cor da paleta do logo
2. Use o glow correspondente
3. Mantenha o contraste adequado
4. Teste em modo claro e escuro
