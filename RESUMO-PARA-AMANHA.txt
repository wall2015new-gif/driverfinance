  💾 BACKUP - MUITO IMPORTANTE!
═══════════════════════════════════════════════════════════════

⚠️ FAÇA BACKUP SEMANAL!

Como fazer:
1. Clique no botão flutuante (⚙️)
2. Exportar → Backup Completo
3. Salve o arquivo JSON no Google Drive/Dropbox

Por quê?
• Os dados ficam salvos no navegador
• Se limpar dados do navegador, perde tudo
• Backup protege seus dados
• Permite transferir entre dispositivos


═══════════════════════════════════════════════════════════════
  📱 DADOS POR DISPOSITIVO
═══════════════════════════════════════════════════════════════

⚠️ IMPORTANTE ENTENDER:

• Cada dispositivo tem seus próprios dados
• Celular e computador NÃO sincronizam
• Chrome e Firefox têm dados separados
• Cada pessoa que acessa tem seus próprios dados

Para usar em múltiplos dispositivos:
1. Faça backup no dispositivo 1
2. Importe o backup no dispositivo 2
