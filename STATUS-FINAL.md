## 🆘 SOLUÇÃO DE PROBLEMAS

### App não atualiza?
```bash
1. Ctrl+Shift+C (limpar cache)
2. Fechar e reabrir app
3. Ou: Ctrl+F5 (hard refresh)
```

### Dados sumiram?
```bash
1. Verificar se está no mesmo navegador
2. Verificar se não limpou dados do navegador
3. Restaurar do backup (se tiver)
```

### Notificações não funcionam?
```bash
1. Verificar permissões do navegador
2. Configurações → Notificações → Permitir
3. Testar com "Modo Motorista"
```

### Gráficos não aparecem?
```bash
1. Verificar conexão (primeira vez precisa baixar Chart.js)
2. Limpar cache e recarregar
3. Verificar console do navegador (F12)
```

### Botão de instalação não some?
```bash
1. Fechar e reabrir app
2. Verificar se realmente instalou
3. Aguardar 5 segundos (verificação automática)
```
